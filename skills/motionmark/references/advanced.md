# Advanced Feature Index

This file is kept as a compatibility router. Prefer the focused references below:

- `composition.md` - scenes, groups, camera, masks, compositing, motion paths, repeat clones, audio, imports, `@draw`, and `@system`.
- `particles.md` - complete `@emitter` syntax and recipes.
- `3d.md` - 2.5D/3D properties, primitives, solid shapes, sort modes, billboards, and depth controls.
- `3d-patterns.md` - deeper 3D examples.
- `paths.md` - path authoring and generated path methods.
- `motions.md` - reusable `@motion` definitions.
- `dsl-syntax.md` - complete syntax and property inventory.

When a task needs more than one advanced feature, read the smallest focused files that match the request.
